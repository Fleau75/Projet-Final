# Access+ App

Application mobile React Native pour les PMR (Personnes à Mobilité Réduite).

## Fonctionnalités

- Authentification Firebase
- Recherche via Google Places API
- Carte interactive avec React Native Maps
- Fiche lieu + avis
- Ajout d’un avis avec photo (Firebase Storage)

## Démarrage

```bash
npm install
npx expo start
```

## Configuration

Ajoutez vos clés Firebase dans `services/firebase.js`  
Ajoutez votre clé Google Maps pour `react-native-maps` et Places API.
